# 개발동기 및 개발 요약
1. 개발동기
  - 회원이 영화관 예매과정에서 영화관에서 주로 앉는 자리를 설문으로 파악 추천해주는 방법이 없을까 하고 진행
2. 요약
  - 영화표 예매를 위해서는 무조건 회원가입이 필요하며, 회원가입과정에서 간단한 설문을 통해 사용자가 주로 어떤 자리를 선호   
    하는지를 파악한다.
